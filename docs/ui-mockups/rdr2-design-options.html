Red Dead Redemption 2 Design Language Implementation Options

Option 1: Parchment & Leather Theme

Traditional Old West aesthetics with parchment textures, leather borders, and warm brown/gold color palette. Emphasizes authenticity and immersion.

Typography

Borders: 2-3px solid, worn edges
Shadows: Subtle, warm tones
Corners: Slightly rounded (4-8px)
Opacity: 0.85-0.98 for overlays

Option 2: Minimal Western Theme

Cleaner, more modern take on Western aesthetics. Retains thematic colors but reduces visual noise. Better for quick readability during gameplay.

When to Use Parchment & Leather

Best For:

  • Players who prioritize immersion
  • Full-screen overlays with space for detail
  • Static/persistent UI elements
  • Storytelling and atmosphere

When to Use Minimal Western

Best For:

  • Players who prioritize performance
  • Quick-glance information during combat
  • Small tooltips and popups
  • Reduced visual clutter

Option 3: Hybrid Approach (RECOMMENDED)

Combines the best of both worlds: rich parchment aesthetics for persistent UI (tracker, status bar), minimal design for transient elements (tooltips, notifications). Balances immersion with usability.

Implementation Checklist

Steps to apply the chosen design language to the QML overlay.

1. Update Theme.qml

Define color palette constants
Create persistent/transient variants
Set typography standards
Define spacing/sizing scales

2. Apply to Components

CollectibleTooltip → Minimal
CollectionTracker → Parchment
StatusPill → Parchment
VideoPlayer → Minimal

3. Font Integration

Download Courier Prime font
Add to qml/fonts/ directory
Register in FontLoader
Set as Theme.fontFamily

4. Testing

Verify readability at 1920x1080
Test with various collectible types
Check color contrast ratios
Validate on actual gameplay footage
